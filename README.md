# API UBS - Instituição Educacional
Este pequeno projeto utilizando-se de Java Spring Boot foi desenvolvido para aplicar na prática os conceitos aprendidos no curso de Java do programa de trainee 2026 da UBS.

# Instructions to Execute the Project:
Abra a pasta raiz do projeto no terminal.

```
mvn spring-boot:run
```

# Instructions to Execute the Tests
Abra a pasta raiz do projeto no terminal.

```
mvn test
```
